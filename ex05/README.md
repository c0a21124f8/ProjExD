# 第5回
## 逃げろ！こうかとん（ex05/dodge.py）
### ゲーム概要
- ex04/dodge_bomb.pyを実行すると，1600x900のcanvasに草原が描画され、こうかとんを移動させて飛び回る爆弾から逃げるゲーム

- こうかとんが爆弾と接触するとゲームオーバーで終了する
### 操作方法
- 矢印キーでこうかとんを上下左右に移動する
### 追加機能
* クリックするごとにスキンが変更できる
* ２のキーを押すとマウスの座標にこうかとんが移動する
